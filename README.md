# Technologie

**Tohle jsem přehodnotil**

> :warning:Webové UI bude postavené na balíkovacím systému **npm** s tím, že se budu snažit využít **REACT**, nebo **Angular**

> :warning:

> - Chtěl jsem se naučit trochu víc s **npm**
